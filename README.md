# Fontes do Vídeo do Canal do Youtube/InfoCotidiano  
## API com JWT (Python + Flask) usando MVC  

## Criar uma API com os recursos:  
Autenticação via client_id e client_secret  
Geração de JWT   
Proteção de endpoints com Bearer Token válido por 1 hora  
Estrutura organizada em MVC  
Uso de variáveis de ambiente com .env  

## Instalação das dependências  
pip install flask pyjwt python-dotenv  

# Arquivo .env  
CLIENT_ID=meu_cliente  
CLIENT_SECRET=segredo_super  
JWT_SECRET=chave_jwt_super_secreta
